# Service Tracker Dashboard

This folder is ready for GitHub Pages.

## Files

- `index.html` - GitHub Pages entry file
- `service-tracker-dashboard.html` - same dashboard file with the descriptive name

## Publish Options

1. Create a GitHub repository.
2. Upload the files from this `outputs` folder to the repository root.
3. In GitHub, go to Settings -> Pages.
4. Set Source to `Deploy from a branch`.
5. Select the branch, usually `main`, and folder `/root`.
6. Save.

GitHub will provide a public URL after the site is deployed.
