# Service Tracker for Android

## Install the APK

1. Download `Service-Tracker-Android.apk` on the Android phone.
2. Open the downloaded file.
3. If Android asks for permission, allow the browser or Files app to install apps from this source.
4. Select **Install**, then open **Service Tracker** from the home screen.

This APK is a test build for direct installation. Android may display an unknown-source warning because it is not distributed through Google Play.

## Install as a PWA

1. Open the live Service Tracker website in Chrome.
2. Open Chrome's menu.
3. Select **Add to Home screen** or **Install app**.

The PWA receives website updates automatically. The APK must be rebuilt and reinstalled when packaged app files change.
